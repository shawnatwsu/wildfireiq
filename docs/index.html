How This Works
WildfireIQ uses a feedforward neural network (7 inputs → 4 hidden → risk score) running entirely in your browser. No data ever leaves your device.

7 Risk Features:
🌡️ Air temperature · 💧 Relative humidity
💨 Wind speed · 🌫️ Vapor pressure deficit
🌧️ 7-day precipitation · 🌱 Soil moisture
📅 Day of year (fire season)
Data Sources
🌤 Open-Meteo API — Weather & forecast
🔥 NASA EONET — Active fire events
🚨 NOAA Weather API — Red Flag Warnings
🗺 Nominatim / OpenStreetMap — Geocoding

Zero signups · Zero API keys · Zero data stored
